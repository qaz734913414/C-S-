# C-S架构的服务器客户端模版
该模版由三部分的程序组成，一个服务端运行的程序，一个客户端运行的程序，还有一个公共的组件，实现了基础的账户管理功能，版本控制，软件升级，公告管理，消息群发功能。具体的操作方法见演示就行。
